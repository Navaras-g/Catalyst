# FocusFlow - Productivity Management Platform

## Project Overview

FocusFlow is a full-stack productivity management platform designed to help users organize tasks, manage goals, build habits, improve focus, and monitor personal productivity through a unified dashboard.

## Core Features (MVP)

### 1. User Management
- Registration
- Login/Logout
- JWT Authentication
- Password Reset
- User Profiles

### 2. Task Management
- Create/Edit/Delete Tasks
- Priority Levels
- Categories and Tags
- Due Dates
- Estimated Time
- Subtasks
- Recurring Tasks
- File Attachments
- Kanban Board

### 3. Project Management
- Create Projects
- Group Tasks into Projects
- Project Progress Tracking
- Deadlines
- Attachments

### 4. Goal Management
- Long-Term Goals
- Milestones
- Goal Progress Tracking
- Completion Metrics

### 5. Habit Tracking
- Habit Creation
- Daily Logging
- Streak Tracking
- Completion Analytics

### 6. Focus & Pomodoro
- Pomodoro Timer
- Focus Sessions
- Focus Statistics
- Task-Linked Sessions

### 7. Calendar System
- Day / Week / Month Views
- Deadlines
- Meetings
- Events
- Drag-and-Drop Scheduling

### 8. Productivity Dashboard
- Tasks Completed
- Focus Hours
- Habit Metrics
- Goal Progress
- Activity Heatmaps
- Trend Charts

### 9. Notes & Knowledge Vault
- Markdown Notes
- Search
- Link Notes to Tasks
- Link Notes to Projects
- Link Notes to Goals

### 10. Notifications & Reminders
- Task Reminders
- Goal Reminders

- Browser Notifications

### 11. Gamification
- XP System
- User Levels

- Streak Rewards

### 12. Real-Time Features
- Live Notifications
- Real-Time Timers
- Instant Dashboard Updates
